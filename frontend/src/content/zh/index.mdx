# 概览

## Mic Service Agent 是什么？

Mic Service Agent（MSA）是一个面向个人与团队协作的中文智能服务工作台。它把模型调用、文件处理、智能体执行、共享空间协作和结果沉淀统一到一个产品中。

## 核心能力

- 多用户与邀请制账号体系
- 个人空间与共享空间协作
- 文件上传、分析、导出与产物沉淀
- 支持多模型接入与图形化模型配置（管理员）
- 支持技能调用、PPT 生成、长任务执行等复杂工作流

## 典型使用示例

### 示例 1：团队共享空间分析
1. 管理员创建共享空间
2. 邀请成员加入
3. 成员上传文件并共同分析
4. 在同一空间中查看对话、产物与结论

### 示例 2：个人空间沉淀
1. 在个人空间中发起研究任务
2. 上传资料并让智能体整理
3. 保存个人记忆与个人智能体
4. 导出报告或产物

### 示例 3：管理员模型配置
1. 打开模型管理中心
2. 添加 OpenAI、MiniMax 等模型
3. 测试连接并设为默认模型
4. 提供给团队成员直接使用

## 使用建议

- 个人资料、个人记忆、个人智能体建议放在个人空间中
- 团队需要共享分析过程和结果时，使用共享空间
- 产物型任务（如 PPT、报告、导出文件）建议在新线程中执行，便于结果整理
