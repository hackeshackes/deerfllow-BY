# v1.6.0 — 画布 + 协作 + 连接器 完整化 实施计划(大纲)

> **For agentic workers:** This is an **outline plan** for a 5-7 week, 5+ person, 3-line-parallel release. Per-task TDD code (steps inside each Task) is **deferred until v1.5.10 is shipped and we re-brainstorm with the v1.5.10 learnings**. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把 v1.5.10 之后剩下的"战略 spec §4.3 P1/P2 + §5 架构演进验证"工作打包为唯一大版本。**v1.6.0 = 真正的架构变化版本**(战略 spec 第 1 段明文)。

**Architecture:** 3 条线并行(团队 5+ 人):

1. **画布后端线** (后端 × 2, 3 周): 完整 Workflow/Node/Edge 域模型 + InMemoryStore + 5 类节点执行器(Agent 委托 LangGraph)+ WorkflowExecutor 顺序拓扑 + VersionManager + FastAPI CRUD
2. **画布前端线** (前端 × 2, 2.5 周): NodeInspector + NodePalette + EdgeConnector + 工作流列表 / 新建 / 详情 / 编辑四页 + 保存 / 运行 / 撤销 / 重做 + 版本对比
3. **协作 + 连接器收尾线** (后端 × 1 + 前端 × 1, 1.5-2 周): PublishService(跨部门发布)+ Slack 连接器 + ABAC 简化版 + 资源隔离(workspace 实际接入 LangGraph runtime)

**Tech Stack:** Python 3.12+ / FastAPI / LangGraph / langchain / Casbin / Next.js 16 / React 19 / React Flow (已装) / Vitest

**Spec:** `docs/superpowers/specs/2026-06-22-micx-ai-portal-roadmap-design.md` §4.3 + §5

**前置:** v1.5.10 发布完成

**预计任务数:** ~25 个任务
**预计总时长:** 5-7 周(5+ 人,3 条线并行)
