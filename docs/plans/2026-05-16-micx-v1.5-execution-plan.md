## 关键风险与缓解

| 风险 | 影响 | 缓解 |
|------|------|------|
| 数据库迁移复杂 | 高 | 提前设计，使用 Alembic 或手动 SQL |
| 向量检索性能 | 中 | 使用 ChromaDB 内置优化，限制结果数量 |
| MCP 预设兼容性 | 中 | 预留自定义配置，支持跳过失败的预设 |
| 渠道连接测试超时 | 低 | 增加超时配置，提供重试机制 |
