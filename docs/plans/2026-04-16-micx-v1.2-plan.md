### 2.2 工具配置完整参数

**当前工具配置结构** (`tool_config.py`):
```python
class ToolConfig(BaseModel):
    name: str           # 工具名称
    group: str         # 分组
    use: str           # 实现类路径
    model_config = ConfigDict(extra="allow")  # 允许额外参数
```

**config.yaml 中的工具**:
```yaml
tools:
  - name: bash
    group: bash
    use: deerflow.sandbox.tools:bash_tool
  - name: read_file
    group: file
    use: deerflow.sandbox.tools:read_file_tool
  - name: write_file
    group: file
    use: deerflow.sandbox.tools:write_file_tool
  # ... 等
```

**extensions_config.json 中的技能启用状态**:
```json
{
  "skills": {
    "image-generation": { "enabled": true },
    "data-analysis": { "enabled": true },
    "ppt-generation": { "enabled": true }
  }
}
```

**问题**: 图像生成、搜索等不是 config.yaml 中的工具，而是技能（Skills）。工具配置 UI 需要区分：
1. **沙箱内置工具** (bash, read_file, write_file, str_replace, ls, grep, glob)
2. **技能类工具** (image-generation, data-analysis, web-search 等)

**修改方案**:
- 扩展 `AdminToolConfig` 类型，增加 `extra_params` 字段
- 前端 UI 增加展开详情，显示各工具的具体参数
