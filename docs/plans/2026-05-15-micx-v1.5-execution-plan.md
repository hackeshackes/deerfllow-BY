#### T1.5: IM 渠道配置 API

**文件**：`backend/app/gateway/routers/channels.py`（新建）

**API 端点**：
```python
GET    /api/channels/config           # 获取渠道配置
PUT    /api/channels/config           # 更新渠道配置
POST   /api/channels/{name}/test      # 测试渠道连接
```

**支持的渠道**：
| 渠道 | 字段 |
|------|------|
| feishu | app_id, app_secret, domain |
| slack | bot_token, app_token, allowed_users |
| telegram | bot_token, allowed_users |
| wecom | bot_id, bot_secret |
| dingtalk | client_id, client_secret |

**验证**：
- [ ] GET /api/channels/config 返回所有渠道配置
- [ ] PUT /api/channels/config 可更新渠道配置
- [ ] POST /api/channels/feishu/test 可测试飞书连接
