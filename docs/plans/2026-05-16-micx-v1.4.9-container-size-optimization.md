## 四、优化方案

### 4.1 方案 A: 替换 sentence-transformers (高优先级)

**问题：** sentence-transformers 带 torch，DeerFlow 使用 CPU 推理

**替代方案：**

#### 方案 A1: 使用 API 方式 embedding（推荐）

完全移除 torch，使用云服务 API：
- OpenAI `text-embedding-3-small` (1536 维)
- Cohere `embed-english-v3.0`
- Jina AI `jina-embeddings-v3`

**优点：**
- 完全移除 torch + transformers + sklearn + scipy（约 700 MB）
- embedding 质量更好
- 无需本地 GPU

**缺点：**
- 需要网络请求
- 可能产生 API 成本
- 离线不可用

#### 方案 A2: 使用 fastembed（轻量方案）

fastembed 是 sentence-transformers 的轻量替代：
- 总大小约 50 MB（vs 600+ MB）
- 使用 ONNX 加速
- 离线可用

**安装：**
```bash
pip install fastembed
```

**代码修改：**
```python
# 替换 sentence-transformers
from fastembed import TextEmbedding
model = TextEmbedding("BAAI/bge-small-en-v1.5")
```

**预期节省：** 550 MB

### 4.2 方案 B: 溯源并移除不需要的包

需要溯源的包：
- `speech_recognition` (40 MB)
- `youtube_transcript_api` (8.6 MB)
- `onnxruntime` (32 MB)

**操作步骤：**
```bash
# 分析每个大包的来源
cd backend && uv tree | grep <package_name> -B10

# 检查实际使用
grep -r "import speech_recognition" backend/
grep -r "import youtube_transcript_api" backend/
grep -r "import onnxruntime" backend/
```

### 4.3 方案 C: 优化 Docker 构建

#### 多阶段构建优化

当前状态：
```dockerfile
FROM python:3.12-slim-bookworm AS runtime  # 精简镜像
```

**问题：** dev 和 langgraph 服务使用 `dev` stage，保留了完整编译环境

**建议：**
- 只在需要调试的服务使用 dev stage
- 生产环境使用 runtime stage

#### Alpine 基础镜像（风险高）

**问题：** Debian → Alpine 可节省 200-300 MB
**风险：** torch 等原生扩展可能需要重新编译

### 4.4 方案 D: 使用 uv 的 slim 模式

```bash
# 使用 --no-dev 和 --strip-only 选项
uv sync --no-dev
uv sync --strip-only
```

## 八、技术细节

### 8.1 当前 embedding 实现

```python
# backend/app/gateway/embedding_service.py

from sentence_transformers import SentenceTransformer

_model: SentenceTransformer | None = None

def _get_embedding_model() -> SentenceTransformer | None:
    global _model
    if _model is None and _model is not False:
        try:
            _model = SentenceTransformer("sentence-transformers/all-MiniLM-L6-v2")
        except Exception as e:
            logger.error(f"Failed to load embedding model: {e}")
            _model = False
    return _model if _model is not False else None

def _generate_embedding(text: str) -> list[float]:
    model = _get_embedding_model()
    if model is not None:
        embedding = model.encode(text)
        return embedding.tolist()
    # Fallback: 使用 hash 生成伪 embedding
    dim = 384
    text_hash = hashlib.sha256(text.encode()).digest()[:dim]
    return [float(b) / 255.0 * 2.0 - 1.0 for b in text_hash]
```

### 8.2 模型信息

| 模型 | 维度 | 大小 | 性能 |
|------|------|------|------|
| all-MiniLM-L6-v2 | 384 | ~90 MB | 中等 |
| bge-small-en-v1.5 (fastembed) | 512 | ~50 MB | 相当 |
