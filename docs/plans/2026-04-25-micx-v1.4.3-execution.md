## 二、Phase 1: 数据库迁移

### 1.1 知识库 is_global 字段

**文件**: `backend/app/gateway/routers/knowledge.py`

```python
# 在 knowledge.py 初始化时执行（仅首次）
ALTER_SQL = """
ALTER TABLE knowledge_bases ADD COLUMN is_global BOOLEAN DEFAULT FALSE;
CREATE INDEX IF NOT EXISTS idx_kb_is_global ON knowledge_bases(is_global);
"""

def _ensure_kb_migration():
    conn = get_db()
    cursor = conn.execute("PRAGMA table_info(knowledge_bases)")
    columns = [row[1] for row in cursor.fetchall()]
    if "is_global" not in columns:
        conn.execute(ALTER_SQL)
        conn.commit()
```

**验证**:
```bash
curl -s http://localhost:2026/api/knowledge | jq '.[].is_global'  # 确认字段存在
```

## 三、Phase 2: 技能可见化改造

### 2.1 Backend - 技能可见性逻辑修改

**文件**: `backend/packages/harness/deerflow/admin/user_skill_store.py`

**函数**: `get_visible_skills_for_user` (约 line 242)

**修改前**:
```python
for skill in all_skills:
    share = shares.get(skill.name)
    if share is None:
        continue  # ← 不可见
    # ...
```

**修改后**:
```python
for skill in all_skills:
    share = shares.get(skill.name)
    if share is None:
        visible.append(skill.name)  # 默认公开
    elif share.owner_id == user_id:
        visible.append(skill.name)
    elif share.visibility == "public":
        visible.append(skill.name)
    elif share.visibility == "workspace" and user_is_in_workspace(user_id, share.workspace_id):
        visible.append(skill.name)
```

### 2.2 Backend - 技能删除权限修改

**文件**: `backend/app/gateway/routers/skills.py`

**函数**: `delete_skill` / `delete_custom_skill`

**修改**: 将 `is_owner` 判断改为：
```python
# admin 可以删除任何技能
if is_owner:
    pass  # 放行
# 普通用户只能删除自己创建的（owner_id == user.id）
elif share and share.owner_id != user.id:
    raise HTTPException(status_code=403, detail="无权删除此技能")
# system: 前缀的技能任何人不得删除
elif skill_name.startswith("system:"):
    raise HTTPException(status_code=403, detail="内置技能不可删除")
```

### 2.3 Backend - 技能 enable/disable 权限

**文件**: `backend/app/gateway/routers/skills.py`

**修改**: `PUT /api/skills/{name}` 或 `POST /api/skills/{name}/enable`

普通用户可以对自己可见的任意技能执行 enable/disable（对自己生效，不影响他人）

### 2.4 验证

